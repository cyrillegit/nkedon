Cet application est une application de gestion d'un magasin.
Cet application comporte plusieurs rubriques, notamment
 - gestion des utilisateurs (creation, modification, suppression)
 - gestion des profils utilisateurs (administrateurs, inspecteurs, inventoristes, caissiers, visiteurs)
 - gestion du magasin
    - gestion des produits
    - gestion des factures
    -gestion des fournisseurs
    - inventaire du magasin
    - statistiques du magasin
 -gestion de la production (en construction)
